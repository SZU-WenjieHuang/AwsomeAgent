## 🚀 推荐的开源 AI Coding 项目

### 1. **Continue.dev** ⭐ 强烈推荐入门
- **GitHub**: [continue.dev](https://github.com/continuedev/continue)
- **特点**: 
  - 完全开源，支持 VS Code 和 JetBrains
  - 可连接任意 LLM（OpenAI、Anthropic、本地模型等）
  - 支持代码补全、聊天、内联编辑
- **适合**: 想了解 AI Coding 工具基础架构的初学者

### 2. **Roo Code** (原 Roo Cline)
- **GitHub**: [RooVetGit/Roo-Code](https://github.com/RooVetGit/Roo-Code)
- **特点**:
  - 开源 AI 编程助手，集成在 VS Code 中
  - 支持多模型、自主编码 Agent
  - 深度理解代码库
- **适合**: 想研究 Agent 模式和多模型支持的开发者

### 3. **Cline**
- **GitHub**: [cline/cline](https://github.com/cline/cline)
- **特点**:
  - 自主编码 Agent，可使用 CLI 和编辑器
  - 支持文件操作、终端命令执行
- **适合**: 研究 AI Agent 与 IDE 集成

### 4. **Aider**
- **GitHub**: [Aider-AI/aider](https://github.com/Aider-AI/aider)
- **特点**:
  - 终端中的结对编程工具
  - 与本地 git 仓库深度集成
  - 支持多文件编辑
- **适合**: 喜欢终端工作流的开发者

### 5. **TabbyML**
- **GitHub**: [TabbyML/tabby](https://github.com/TabbyML/tabby)
- **特点**:
  - 自托管的开源代码助手
  - 支持代码补全
- **适合**: 研究私有化部署

## 📚 学习 Roadmap

### **阶段 1: 基础理解 (1-2 周)**
1. **了解 AI Coding 的核心概念**
   - Code Completion (代码补全)
   - Inline Chat (内联聊天)
   - Agent Mode (Agent 模式，自主执行多步骤任务)
   - Context Awareness (上下文感知)

2. **阅读 Continue.dev 源码**
   - 从简单的 VS Code Extension 架构入手
   - 理解如何与 LLM API 交互
   - 学习如何处理代码上下文

### **阶段 2: 深入 Agent 架构 (2-4 周)**
1. **研究 Roo Code / Cline 的 Agent 实现**
   - 工具调用 (Tool Calling) 机制
   - 多轮对话管理
   - 文件系统操作
   - 终端命令执行

2. **关键组件学习**
   - **MCP (Model Context Protocol)**: 标准化 AI 工具接口
   - **LSP (Language Server Protocol)**: 代码分析基础
   - **RAG (Retrieval-Augmented Generation)**: 代码库检索

### **阶段 3: 实践与扩展 (持续)**
1. **动手实现简单原型**
   - 基于 VS Code Extension API 做一个基础插件
   - 集成 OpenAI/Claude API
   - 实现代码补全和聊天功能

2. **研究高级特性**
   - 多文件编辑 (Multi-file Editing)
   - 代码库索引和搜索
   - 长期记忆机制
   - 安全沙箱执行
