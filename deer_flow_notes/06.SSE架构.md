### 4.3 代码层面的流式处理

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                              后端流式生成代码 (client.py)                                   │
├─────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                         │
│   class DeerFlowClient:                                                                 │
│       def stream(self, message, *, thread_id=None, **kwargs):                          │
│           """生成流式事件"""                                                            │
│                                                                                         │
│           # 1. 确保 Agent 已创建                                                         │
│           self._ensure_agent(config)                                                    │
│                                                                                         │
│           # 2. 调用 Agent 的流式执行                                                     │
│           for chunk in self._agent.stream(                                              │
│               state,                                                                    │
│               config=config,                                                            │
│               context=context,                                                          │
│               stream_mode="values"               # 请求流式输出                        │
│           ):                                                                            │
│               messages = chunk.get("messages", [])                                      │
│                                                                                         │
│               # 3. 遍历消息，生成 SSE 事件                                                │
│               for msg in messages:                                                      │
│                   if isinstance(msg, AIMessage):                                        │
│                       # AI 消息事件                                                      │
│                       yield StreamEvent(                                                │
│                           type="messages-tuple",                                        │
│                           data={"type": "ai", "content": text, ...}                     │
│                       )                                                                 │
│                                                                                         │
│                   elif isinstance(msg, ToolMessage):                                    │
│                       # 工具执行结果事件                                                 │
│                       yield StreamEvent(                                                │
│                           type="messages-tuple",                                        │
│                           data={"type": "tool", "content": result, ...}                 │
│                       )                                                                 │
│                                                                                         │
│               # 4. 生成状态快照事件                                                       │
│               yield StreamEvent(                                                        │
│                   type="values",                                                        │
│                   data={"title": ..., "messages": ..., "artifacts": ...}                │
│               )                                                                         │
│                                                                                         │
│           # 5. 生成结束事件                                                              │
│           yield StreamEvent(type="end", data={"usage": ...})                            │
│                                                                                         │

│   │   - end: 流结束 + Token 统计                                                      │   │
│   │                                                                                 │   │
│   └─────────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                         │
│   关键区别：                                                                             │
│   - messages-tuple 是"消息视角"（谁说了什么）                                           │
│   - values 是"状态视角"（当前完整状态）                                                 │
│   - on_tool_end 是"执行视角"（工具怎么执行的）                                           │
│                                                                                         │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```
