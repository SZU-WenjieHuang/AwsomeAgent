# 开源 Agent 项目推荐

## 值得学习的 Agents 项目

### 1. Hermes Agent
- **GitHub**: https://github.com/reworkd/AgentGPT
- **简介**: 基于 LangChain 的自主 AI Agent 平台，支持浏览器自动化、代码执行等能力
- **特点**: 模块化设计、支持多种 LLM、可视化任务流

### 2. OpenClaw (OpenManus)
- **GitHub**: https://github.com/openclaw/openclaw
- **简介**: 开源的通用 AI Agent 框架，支持多工具调用和自主任务执行
- **特点**: 轻量级、易于扩展、支持浏览器自动化和代码执行

### 3. DeerFlow
- **GitHub**: https://github.com/bytedance/deer-flow
- **简介**: 字节跳动开源的 AI Agent 框架，专注于复杂任务分解和执行
- **特点**: 强大的任务规划能力、多 Agent 协作、企业级稳定性

### 4. AutoGPT
- **GitHub**: https://github.com/Significant-Gravitas/AutoGPT
- **简介**: 最早的自主 AI Agent 项目之一，支持长期记忆和自主决策
- **特点**: 社区活跃、插件丰富、支持多种任务类型

### 5. CrewAI
- **GitHub**: https://github.com/joaomdmoura/crewAI
- **简介**: 多 Agent 编排框架，专注于 Agent 团队协作
- **特点**: 简单易用、支持角色定义、任务委派清晰

### 6. LangGraph
- **GitHub**: https://github.com/langchain-ai/langgraph
- **简介**: LangChain 官方推出的 Agent 状态机框架
- **特点**: 可视化工作流、循环支持、生产级可靠性

### 7. Dify
- **GitHub**: https://github.com/langgenius/dify
- **简介**: LLM 应用开发平台，支持 Agent 和工作流编排
- **特点**: 可视化界面、支持多种模型、易于部署
